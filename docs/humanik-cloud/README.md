# Humanik Cloud Integration

This worker includes optional integration with [Humanik Cloud](https://humanik.io) — an AI-powered cloud orchestration platform. These features activate automatically when Humanik Cloud environment variables are present and are completely inert when they're not.

**If you're using this worker standalone (without Humanik Cloud), you can ignore this entire section.** The worker runs as a standard OpenAI-compatible vLLM server with no external dependencies.

## What Humanik Cloud Adds

| Feature | What it does | Without Humanik Cloud |
|---------|-------------|----------------------|
| [HMAC Auth](hmac-auth.md) | Gates all inference endpoints behind control plane signature verification | Middleware still runs but returns 500 if no secret is configured — set `CP_INSTANCE_HMAC_SECRET` or remove the middleware for standalone use |
| [Heartbeat](heartbeat.md) | Reports instance health, load, and routing info to Redis for the ALB | Silently disabled — no Redis calls, no errors |
| [Idle Timeout](idle-timeout.md) | Auto-terminates the pod after configurable idle period to save costs | Silently disabled — pod runs indefinitely |
| Pod Lifecycle | Control plane creates/destroys pods, injects env vars, routes traffic | You manage the pod yourself |

## Environment Variables

All Humanik Cloud features are driven by environment variables injected by the control plane at pod creation. None are required for standalone operation.

```bash
# ── Identity (required for heartbeat + HMAC) ──────────────────
NEXUS_TENANT_ID=tenant_abc123        # Your Humanik Cloud tenant
NEXUS_SERVICE_ID=svc_gemma           # Service identifier
NEXUS_INSTANCE_ID=inst_a1b2c3d4e5f6  # Unique instance ID (generated by CP)
NEXUS_PROVIDER=runpod                # Provider type
NEXUS_REDIS_URL=redis://host:6379    # Redis for heartbeat + termination signals

# ── Security ──────────────────────────────────────────────────
CP_INSTANCE_HMAC_SECRET=<shared-secret>  # HMAC signing secret

# ── Routing ───────────────────────────────────────────────────
RUNPOD_POD_ID=abc123xyz              # Auto-detected from hostname if not set
RUNPOD_DC_ID=US-TX-3                 # Datacenter region

# ── Scaling ───────────────────────────────────────────────────
IDLE_TIMEOUT_MINUTES=20              # 0 = persistent (never auto-terminate)
```

## Architecture

```
User → Humanik Cloud API (auth, quota, rate limiting)
  → Control Plane (ALB routing, scaling decisions)
    → HMAC-signs request
      → This Worker (pod:8000)
        → Verifies HMAC
        → vLLM inference
        → Response back through the chain

Meanwhile:
  Worker → Redis heartbeat every 10s (status, load, routing info)
  Control Plane ← reads heartbeats for ALB decisions
  Worker → Redis pub/sub termination signal on idle/shutdown
  Control Plane ← destroys pod on termination signal
```
